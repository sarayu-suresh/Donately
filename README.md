# Donately

## Description

A web-based app  to assist in collecting the leftover food from hotels &amp; restaurants and other useful items from donors.


## Getting Started

### Dependencies

* Flask==2.0.3
* Flask_Bcrypt==1.0.1
* Flask_Login==0.6.1
* Flask_Mail==0.9.1
* Flask_SQLAlchemy==2.5.1
* Flask_WTF==1.0.1
* itsdangerous==2.0.1
* Pillow==9.2.0
* WTForms==3.0.1

### Installing

* Clone the repo
* Install all dependencies

### Executing program

```
python run.py
```
